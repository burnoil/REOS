# Rockwell identity assets

REOS shall use a verified historical Rockwell corporate logo asset rather than a typed approximation.

## Required artwork

- Primary horizontal Rockwell wordmark
- Corporate symbol/mark where historically appropriate
- Monochrome dark version for equipment-white surfaces
- Reversed white version for Mission Blue fields

## Implementation rules

- Preserve original proportions.
- Do not redraw the wordmark with a generic system font.
- Do not stretch, condense, outline, bevel, shadow, or otherwise stylize the mark.
- Maintain clear space between the Rockwell identity and REOS system text.
- Store approved source artwork in this directory and rendered Rainmeter assets under `Rainmeter/@Resources/Images/`.

The current Header Assembly reserves a dedicated identity area and uses a temporary text fallback until verified artwork is committed.
